Seven nodes. Zero code. Under 15 minutes from import to first auto-reply. Here's exactly what each piece does and how to configure it.

Node 1: Gmail Trigger

Purpose

Watches your inbox for new unread emails. Polls every 5 minutes by default.

Configuration ▸ Operation: Trigger on Incoming Email ▸ Filter: Unread only ▸ Polling: Every 5 minutes ▸ Mark as read: After processing ✓ ▸ Pro tip: Create a Gmail filter → label "auto-reply" → trigger watches only that label for total control

Node 2: OpenAI Analyzer

Purpose

Reads each email like a human would. Understands context, intent, sentiment, urgency. Outputs structured data for routing decisions.

Model Settings ▸ Model: gpt-4o-mini (fast, cheap, accurate enough) ▸ Temperature: 0.1 (consistent results) ▸ Max tokens: 300 ▸ Cost: ~$0.0003 per email ($0.30 per 1,000)
System Prompt (Core Logic)

Categorize into ONE of: FAQ, SUPPORT, CONFIRMATION, INFO_REQUEST, MEETING_REQUEST, UNSUBSCRIBE, SPAM/PROMO, NEEDS_HUMAN

Output JSON with: category, confidence, sentiment, urgency, key_topics, suggested_response_type

Node 3: Switch Router

Purpose

Traffic controller. Routes each email down the right path based on AI categorization. 8 possible routes.

Routing Map
FAQ → Template A
SUPPORT → Template B + Flag
CONFIRM → Template C
MEETING → Template D
NEEDS_HUMAN → Skip Reply → Alert You

Node 4: Template Builder

Purpose

Generates personalized reply from template library. Inserts sender name, references their question, matches detected tone.

Each category has its own template. See Page 04 for the complete template library ready to copy-paste.

Node 5: Gmail Sender

Purpose

Sends the generated reply. Looks like you typed it yourself. Marks original as replied.

Send Configuration ▸ From: Your Gmail ▸ To: Original sender ▸ Reply-To: Your actual email (replies come to you) ▸ Label applied: "Auto-Replied" (for tracking)

Node 6: Google Sheets Logger

Purpose

Full audit trail. Every email logged with timestamp, sender, subject, category, sentiment, whether it was auto-replied or escalated.

Node 7: Slack/Discord Alert (Optional)

Smart Escalation

Only fires when: category = NEEDS_HUMAN OR urgency = high. You get pinged for what actually matters. Everything else? Handled silently.

Page 04 Reply Templates

Templates That Actually Convert

These aren't robotic auto-responses. They're crafted to feel personal, helpful, and human — because the best automation is invisible. Recipients should never know a machine wrote this unless you tell them.

[Visual: Dark bg. Email mockup cards floating, each with different colored left border (green=FAQ, cyan=confirm, amber=support). Subtle glow. "Feels human" badge on each. Premium email client aesthetic.]

Template A: FAQ Response

Use When: Pricing, availability, how-it-works questions

Subject: Re: {{ original_subject }}

Hi {{ sender_name }},

Great question! Here's the quick answer:

{{ dynamic_answer_based_on_detected_topic }}

Want to dive deeper? Just reply to this email—I'm happy to chat!

Best,
[Your Name]

P.S. This message was sent by my automated assistant. If you need personal attention, just reply "HUMAN" and I'll take over personally.

Template B: Support/Empathy

Use When: Complaints, issues, problems, refund requests

Subject: Re: {{ original_subject }}

Hi {{ sender_name }},

I'm really sorry to hear about this issue—that's definitely not the experience we want for you.

Let me look into this right away. In the meantime: {{ common_fix_for_issue_type }}

Your ticket ID: #{{ ticket_number }}
I'll follow up by {{ tomorrow_or_EOD }} with an update.

Thank you for your patience,
[Your Name]

Template C: Quick Confirmation

Use When: "Got it", "received", acknowledgments needed

Subject: Re: {{ original_subject }}

Got it! ✅

Thanks for reaching out, {{ sender_name }}. I've received your message and added it to my queue.

Current response time: within 24 hours.

Talk soon!
[Your Name]

Template D: Info Request

Use When: Wants more details, case studies, portfolio, examples

Subject: Re: {{ original_subject }} — Here's What You Asked For

Hey {{ sender_name }},

Love that you're digging deeper! Here are some resources:

📄 [Case Study / Portfolio Link]
🎥 [Video Demo / Explanation]
📊 [Pricing Page]
❓ [FAQ Page]

Which of these is most interesting to you? Happy to elaborate.

Cheers,
[Your Name]

Template E: Meeting Request

Use When: Wants to schedule call, demo, consultation

Subject: Schedule Time — Let's Make It Happen! 📅

Hi {{ sender_name }},

Would love to connect!

Here's my calendar—pick a time that works:
[Calendly/Calendar Link]

A few things that help me prepare:
• What's the main topic?
• How much time do you need?
• Anyone else joining?

See you soon!
[Your Name]

Template F: Unsubscribe

Use When: Wants off newsletter, remove from list

Subject: Sorry to See You Go 😔

Hi {{ sender_name }},

No problem at all—you're now removed from our list.

Was there something specific that didn't click? I genuinely read every response and use feedback to improve.

Either way, wish you the best!

Take care,
[Your Name]

◆
Customization Rules
  • Edit these to match YOUR voice. If you're casual, keep them casual. If formal, adjust accordingly.
  • The P.S. line is optional but recommended. Transparency builds trust. Most people think it's cool, not weird.
  • Always include a human escape hatch. "Reply HUMAN" or similar lets frustrated users bypass automation.
  • Test each template by sending to yourself first. Read it as if you received it cold. Does it feel real?
