Step-by-Step Build Guide

📋 Prerequisites (5 minutes)

  • n8n account (free cloud or self-hosted)
  • Gmail account (for sending emails)
  • Google Sheet (create blank sheet named "Lead Tracker")
  • Webhook URL from your form (or use test data below)
1
Webhook Trigger (Receives Form Data)
HTTP Method POST
Path new-lead
Response Code 200

What this does: Waits for data from your client's contact form. Every time someone submits, n8n wakes up and grabs the data instantly.

💡 Pro Tip: Test this first using Postman or curl. Send test data:
JSON 
{
  "name": "Test User",
  "email": "test@example.com",
  "phone": "555-1234",
  "source": "website"
}
2
Google Sheets (Log the Lead)
Operation Append
Sheet Lead Tracker

Columns to map:

Column Mapping 
Column A: Timestamp → {{ $now.format('YYYY-MM-DD HH:mm') }}
Column B: Name      → {{ $json.body.name }}
Column C: Email     → {{ $json.body.email }}
Column D: Phone     → {{ $json.body.phone }}
Column E: Source    → {{ $json.body.source }}
Column F: Status    → "New"
Why this matters: Clients LOVE seeing their data organized. This alone looks professional AF.
3
Gmail / SendGrid (Welcome Email)
Operation Send
To {{ $json.body.email }}

Subject Line:

Hey {{ $json.body.name.split(' ')[0] }}, welcome to [Business Name]! 🎉

Email Body (HTML):

HTML Template 
<h2>Hi {{ $json.body.name.split(' ')[0] }},</h2>

<p>Thanks for reaching out! I got your message 
and I'm excited to help.</p>

<p>Here's what happens next:</p>
<ul>
  <li>I'll review your inquiry within 24 hours</li>
  <li>You'll get a personalized response 
      (not a bot—I promise 😄)</li>
  <li>If it's urgent, just reply to this email</li>
</ul>

<p>Talk soon!<br>
<strong>[Your/Client Name]</strong></p>

<p><small>P.S. Quick question—what's the #1 thing 
you're hoping we can help with?</small></p>
🧠 PSYCHOLOGY HACK: That P.S. question? It doubles response rates. Seriously. Use it every time.
4
Wait Node (The 72-Hour Delay)
Amount 72
Unit Hours

What this does: Pauses the workflow for exactly 3 days. The lead gets time to breathe, then...

5
IF Node (Smart Follow-up Logic)

Condition:

Check if lead opened welcome email

If using SendGrid:
  Check webhook event = "open"

If no email tracking available:
  SKIP this node → always send follow-up
Why: Only follow up with engaged leads. Don't spam people who didn't open email #1. This keeps your sender reputation clean.
6
Second Email (Follow-Up)

Subject:

Quick check-in, {{ $json.body.name.split(' ')[0] }} 👋

Body:

HTML Template 
<h2>Hey again!</h2>

<p>I wanted to follow up and see if you had 
a chance to think about things.</p>

<p>No pressure at all—but if you have any 
questions or want to move forward, 
just hit reply.</p>

<p>If now's not the right timing, totally 
understand. I'm here when you're ready.</p>

<p>Cheers,<br>[Name]</p>
TONE NOTE: Casual, low-pressure. Not salesy. Humans respond to humans. This isn't a funnel—it's a conversation.
7
Google Sheets (Update Status)
Operation Update
Find by Email address
Status → "Followed Up"
Last Contact → {{ $now.format('YYYY-MM-DD') }}

Result: Client opens their Sheet and sees exactly where every lead is at. Beautiful. Professional. Done.

Bonus: Google API Credentials Mini-Guide

Because I know this part sucks.

For Gmail:

  1. Go to Google Cloud Console
  2. Create project → Enable Gmail API
  3. Create OAuth 2.0 credentials
  4. Download JSON → Keep it secret 🔒
  5. In n8n: Credentials → Google Gmail → Paste

For Google Sheets:

  1. Same process as above
  2. But enable "Google Sheets API" instead
  3. Use same OAuth credentials (reuse!) ♻️
⚡ Pro tip: Create ONE service account for both APIs. Saves time, fewer credentials to manage.
