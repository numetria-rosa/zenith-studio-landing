Page 04 Build Guide

Node-by-Node:
Exact Configuration

No vague instructions. Every setting. Every value. Copy this, import the JSON, connect your credentials, and you're live.

⏱ Prerequisites (5 Minutes)

  • ✓
    n8n account (free cloud or self-hosted)

  • ✓
    Google Sheet named "Lead Tracker"

Node 1: Webhook Trigger

Receives Form Data

HTTP Method: POST  |  Path: new-lead  |  Response: 200 OK

Waits for data from client's contact form. Every submission wakes up n8n instantly.

Test Payload $ curl -X POST your-webhook/new-lead \ -H "Content-Type: application/json" \ -d '{"name":"Test User","email":"test@example.com","phone":"555-1234","source":"website"}' → {"status": "received", "message": "Lead captured! 🎯"}

Node 2: Google Sheets (Log Lead)

Append New Row

Operation: Append  |  Sheet: Lead Tracker

Column Mapping Column A: Timestamp → {{ $now.format('YYYY-MM-DD HH:mm') }} Column B: Name → {{ $json.body.name }} Column C: Email → {{ $json.body.email }} Column D: Phone → {{ $json.body.phone }} Column E: Source → {{ $json.body.source }} Column F: Status → "New"
💡 Why this matters: Clients LOVE seeing their data organized. This alone looks professional AF.

Node 3: Gmail (Welcome Email)

Send Personalized Welcome

To: {{ $json.body.email }}  |  Operation: Send

Subject Line Hey {{ $json.body.name.split(' ')[0] }}, welcome! 🎉
🧠 PSYCHOLOGY HACK: Include a P.S. question like "What's the #1 thing you're hoping we can help with?" — It doubles response rates.

Nodes 4-7: Wait → IF → Follow-Up → Update

The Automation Loop
  • Node 4: Wait 72 hours (3-day delay)
  • Node 5: IF check — only follow up engaged leads
  • Node 6: Send casual follow-up email
  • Node 7: Update Sheet status → "Followed Up"

Page 05 Quick Win

Your 20-Minute Path to Proof

No theory. No prep work. Import, connect, customize, test, activate. By the time you finish reading this page, you could have a working workflow.

The Timeline

  1. 0-3 min
    Import workflow JSON into n8n. Click "+" → "Import from File" → Upload. Done.
  2. 3-6 min
    Connect Gmail credential. OAuth flow takes 2 minutes. Grant once, forget forever.
  3. 6-9 min
    Connect Google Sheets credential. Blank sheet. Name it. Connect. 3 minutes max.
  4. 9-14 min
    Customize email templates. Change [Your Name], add links, tweak tone. Start with welcome email.
  5. 14-17 min
    Send test email from alternate address. Subject: "Question about pricing". Wait for magic.
  6. 17-20 min
    Verify response arrived. Check Sheet log. Activate workflow. Toggle ON. Your first automated system is live.
